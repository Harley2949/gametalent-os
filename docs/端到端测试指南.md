## 🐛 常见问题

### 问题 1: 后端连接失败
**解决方案**：
```bash
# 检查后端是否运行
curl http://localhost:3006/api/auth/login

# 检查数据库
docker ps | grep postgres
```

### 问题 2: 登录后跳转 404
**解决方案**：检查 middleware.ts 是否正确配置

### 问题 3: 图表不显示
**解决方案**：检查浏览器控制台是否有 API 错误
