# 邮件通知配置指南

## 📧 概述

GameTalent OS 邮件通知系统支持以下功能：
- ✅ 面试邀请邮件
- ✅ 面试提醒邮件（24小时前、1小时前）
- ✅ Offer 邮件
- ✅ 职位关闭通知

## 🔧 SMTP 服务配置

### 方式 1: Gmail（推荐）

**优点**：稳定、免费、安全
**缺点**：需要开启应用专用密码

#### 配置步骤：

1. **开启 Gmail 两步验证**
   - 访问：https://myaccount.google.com/security
   - 找到"两步验证"并开启

2. **生成应用专用密码**
   - 访问：https://myaccount.google.com/apppasswords
   - 选择"邮件"和"其他（自定义名称）"
   - 点击"生成"，复制 16 位密码（不含空格）

3. **配置 .env 文件**
   ```bash
   SMTP_HOST="smtp.gmail.com"
   SMTP_PORT=587
   SMTP_USER="your-email@gmail.com"
   SMTP_PASS="your-16-char-app-password"
   ```

---

### 方式 2: QQ 邮箱

**优点**：国内访问稳定、免费
**缺点**：需要开启 SMTP 服务

#### 配置步骤：

1. **开启 SMTP 服务**
   - 登录 QQ 邮箱
   - 进入"设置" → "账户"
   - 找到"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"
   - 开启"POP3/SMTP服务"或"IMAP/SMTP服务"
   - 按照提示发送短信验证
   - **保存授权码**（不是 QQ 密码！）

2. **配置 .env 文件**
   ```bash
   SMTP_HOST="smtp.qq.com"
   SMTP_PORT=587
   SMTP_USER="your-email@qq.com"
   SMTP_PASS="your-authorization-code"
   ```

---

### 方式 3: 163 邮箱

**优点**：国内访问稳定、免费
**缺点**：需要开启 SMTP 服务

#### 配置步骤：

1. **开启 SMTP 服务**
   - 登录 163 邮箱
   - 进入"设置" → "POP3/SMTP/IMAP"
   - 开启"SMTP服务"
   - **保存授权码**（不是登录密码！）

2. **配置 .env 文件**
   ```bash
   SMTP_HOST="smtp.163.com"
   SMTP_PORT=465
   SMTP_USER="your-email@163.com"
   SMTP_PASS="your-authorization-code"
   ```

---

### 方式 4: 企业邮箱（阿里云、腾讯云）

#### 阿里云邮件推送
```bash
SMTP_HOST="smtpdm.aliyun.com"
SMTP_PORT=80 或 465
SMTP_USER="your-username@your-domain.com"
SMTP_PASS="your-smtp-password"
```

#### 腾讯企业邮箱
```bash
SMTP_HOST="smtp.exmail.qq.com"
SMTP_PORT=465
SMTP_USER="your-email@your-domain.com"
SMTP_PASS="your-password"
```

## 📋 常见问题

### 1. 认证失败（Authentication failed）

**可能原因**：
- SMTP 密码错误（Gmail 必须使用应用专用密码）
- 用户名或邮箱地址错误
- SMTP 服务未开启

**解决方法**：
- 检查 `SMTP_USER` 是否与邮箱地址一致
- Gmail 用户请重新生成应用专用密码
- QQ/163 邮箱用户请确认已开启 SMTP 服务

### 2. 连接超时（Connection timeout）

**可能原因**：
- SMTP 主机地址错误
- 防火墙阻止了 SMTP 端口
- 网络问题

**解决方法**：
- 检查 `SMTP_HOST` 和 `SMTP_PORT` 是否正确
- 尝试更换端口（587 → 465，或反之）
- 检查防火墙设置

### 3. 邮件进入垃圾箱

**可能原因**：
- 发件人地址未经验证
- 邮件内容被识别为垃圾邮件

**解决方法**：
- 在收件箱中将发件人添加到白名单
- 使用企业邮箱或经过验证的域名

## 🚀 生产环境建议

### 使用专业邮件服务

对于生产环境，建议使用以下专业邮件服务：
1. **SendGrid**（推荐，免费 100 封/天）
2. **阿里云邮件推送**
3. **腾讯云邮件服务**
4. **Amazon SES**

这些服务提供：
- 更高的送达率
- 邮件模板管理
- 发送统计和分析
- 防止被标记为垃圾邮件

## 📝 开发与调试

### 禁用邮件发送（开发环境）

如果在开发时不希望发送真实邮件，可以：
1. 不配置 `SMTP_USER` 和 `SMTP_PASS`
2. 系统会自动跳过邮件发送，不影响业务流程
